Quick Tutorial

Define the options

Define the target path with Edit > Preferences (F10).

Then, define the action corresponding to each button:

  • •Keep  

  • •Zap 

  • •Move  

 

Click Apply.

 

Open the media files

Drag-n-drop the media files into the playlist.

 

Play

Play the first media

Click Keep or Zap or Move.

 

Save the current playlist

The current playlist can be auto-saved, in the default Music directory.

You can also save/open it manually.
